# CI pipeline 

- Lint - check code quality
- Test - check everything is working fine.
- Build - compile a whole project and ready for deployment

for setup locally

# install dependencies
    - npm install

# Run Lint
    - npm run lint

# Run Test
    - npm test

# Run build 
    - npm run build

# Start application
    - npm  run start


# Usage CI/CD for continous Integration and deployment
